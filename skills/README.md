# Skills del proyecto — Backend

Skills de Cursor para flujos repetibles. Ubicacion: `.cursor/skills/`

| Skill | Cuando usar |
| ----- | ----------- |
| [create-module](../.cursor/skills/create-module/SKILL.md) | Crear o completar un modulo en `src/modules/` |
| [create-endpoint](../.cursor/skills/create-endpoint/SKILL.md) | Agregar endpoint a modulo existente |

## Como invocar

En Cursor, referenciar el skill por nombre o pedir al agente que lo lea antes de implementar.

Ejemplo: "Usa el skill create-module para implementar el modulo users"
